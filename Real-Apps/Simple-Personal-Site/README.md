#### Create enviroment
- `python3 -m venv env`

**Active**
- `source env/bin/activate`

**Deactivae**
- `deactivate`

**Update pip**
- `pip install --upgrade pip`
- `pip list`

**Install Flask**
- `pip install flask`

#### App structure
- `mkdir app`
- `pip freeze > requirements.txt`
- For installing packages: `pip install -r requirements.txt`

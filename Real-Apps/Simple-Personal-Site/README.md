#### Create enviroment
- `python3 -m venv env-simple`

**Active**
- `source env-simple/bin/activate`

**Update pip**
- `pip install --upgrade pip`
- `pip list`

**Install Flask**
- `pip install flask`

#### App structure
- `mkdir app`
- `pip freeze > requirements.txt`
- For installing packages: `pip install -r requirements.txt`
